# SkillTree

Interactive skill-tree visualization built with React + TypeScript + Vite.

## Commands

- `npm install` — install dependencies
- `npm run dev` — start Vite dev server (default: http://localhost:5173)
- `npm run build` — production build to `dist/`
- `npm run preview` — preview the production build locally
- `npm run lint` — run ESLint

## Project structure

- `src/skilltree-v2.tsx` — main SkillTree component (all-in-one, inline styles)
- `src/App.tsx` — app wrapper that renders SkillTreeV2
- `src/main.tsx` — React entry point
- `index.html` — HTML shell (Google Fonts loaded here)
